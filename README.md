### Ask an Agent

GitHub profile READMEs cannot run an in-place JavaScript chatbox inside an image, so the office above is the GitHub-safe animated SVG preview.

Open the live room to click an agent and get the chatbox:

[Launch the interactive agent office](https://rjscripts-24.github.io/RJScripts-24/office/interactive-office.html)

The chatbox creates a correctly labelled GitHub Issue, then the GitHub Actions agent pipeline replies and closes it.
